필터 탐색

		eq() : 선택한 엘리먼트 객체들 중에서 인덱스로 탐색
		slice() : 선택한 엘리먼트 객체들 중에서 인덱스범위로 탐색
		first() : 선택한 엘리먼트 객체들 중에서 첫번째 요소를 탐색
		last() : 선택한 엘리먼트 객체들 중에서 마지막 요소를 탐색

트리 탐색

		find() : 선택한 엘리먼트의 자손들 중에 탐색
		children() : 선택한 엘리먼트의 자식 요소들을 탐색
		parent()/parents() : 선택한 엘리먼트의 부모요소 탐색 / 선택한 엘리먼트의 부모요소를 원하는게 나올때까지??탐색뭐어쩌구
		prev()/prevAll() : 바로 앞에 오는 형제요소 선택 / 앞에 있는 모든 형제요소를 탐색
		next()/nwxtAll() : 바로 뒤에 오는 형제요소 선택 / 뒤에 있는 모든 형제요소를 탐색
		siblings() : 선택한 엘리먼트의 형제요소를 모두 탐색한다.
